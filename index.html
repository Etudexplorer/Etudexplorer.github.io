Comment utiliser la carte ?

La carte vous indique différents points d'intérêts qui pourraient vous être utile en tant qu'étudiant.
Vous avez accès à des filtres qui vous permettent de n'afficher que ce qui vous intéresse.
Cliquez sur un point d'intérêt pour afficher toutes les informations sur ce dernier.
